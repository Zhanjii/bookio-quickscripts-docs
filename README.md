## About

BookIO QuickScripts is a desktop application that streamlines eBook production workflows with automated image processing, file management, and more.

### Features

- **Book Ingest** - Process and organize book files
- **Scene Ingest** - Manage scene files efficiently
- **Format Download** - Download files in various formats
- **ZIP Creator** - Package files for distribution
- **New Project** - Initialize new projects with proper structure
- **Diagnostics** - System health checks and troubleshooting
